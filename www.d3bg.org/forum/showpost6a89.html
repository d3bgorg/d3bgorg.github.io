Вариант 1:
1. Свали си ЕU клиента без да изтриваш US клиента!
2. Инсталирай диаблото в същата директория
3. Чакаш 10-15 минути
4. Играеш!

Вариант 2:
This will fix your client to avoid all the downloading and frustration if you cant login due a patch from US.

1. Go in your Diablo III folder.
2. Open the .agent.db file on WordPad.
3. find the "patch_url" line and change its value to:
Код:
http://enGB.patch.battle.net:1119/patch


4. Save and close the file.
5. Now delete the Agent folder in the Battle.net folder: (don't skip this step)

Windows 7
Код:
C:\ProgramData\Battle.net


Windows XP
Код:
C:\Document and settings\All Users\Application Data\Battle.net


(Note that ProgramData and ApplicationData are a hidden folder.)

6. Delete the Updates folder. (Diablo III\Updates)
7. Move the Diablo III.exe file out of the Diablo III folder.
8. Run Diablo III Launcher.
